# プロダクトコンテキスト

## プロダクト概要
Revengerは、ブログ記事やメモを管理・共有するためのWebアプリケーションです。ユーザーは投稿を作成、編集、検索することができます。

## 主要機能
### ユーザー管理
- ユーザー登録・ログイン・ログアウト（Deviseを使用）
- ユーザープロフィール編集
- パスワードリセット

### 投稿管理
- 投稿の作成
- 投稿の編集（インラインでのJavaScript編集）
- 投稿の削除（確認ダイアログあり）
- 投稿の一覧表示

### 検索と並べ替え
- 投稿の検索機能
- 作成日時・更新日時による並べ替え
- 昇順・降順の切り替え

### リマインダー
- リマインダー設定
- メール通知

## ユーザーエクスペリエンス
- シンプルで使いやすいインターフェース
- レスポンシブデザイン
- JavaScriptを使用した動的なUI

## ターゲットユーザー
- ブログ記事やメモを管理したいユーザー
- チーム内で情報を共有したいユーザー

## ユースケース
1. ユーザーが新しい投稿を作成する
2. ユーザーが既存の投稿を編集する
3. ユーザーが投稿を検索する
4. ユーザーが投稿を並べ替える
5. ユーザーが投稿を削除する
6. ユーザーがリマインダーを設定する
